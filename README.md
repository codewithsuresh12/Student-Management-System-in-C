# 🎓 Student Management System in C

A simple and efficient **Student Management System** developed in **C Programming Language**. This project allows users to manage student records with features like adding, displaying, searching, updating, deleting, and storing data in files.

---

## 📌 Features

✅ Add Student Records <br>
✅ Display All Students<br>
✅ Search Student by Roll Number<br>
✅ Update Student Information<br>
✅ Delete Student Record<br>
✅ Save Data to File<br>
✅ Load Data from File Automatically<br>
✅ Menu Driven Interface<br>

## 📖 How It Works

### 1. Add Student

Enter:

* Roll Number
* Name
* Marks

The student record is stored in memory.

### 2. Display Students

Displays all stored student records.

### 3. Search Student

Search a student using their Roll Number.

### 4. Update Student

Modify the student's Name and Marks.

### 5. Delete Student

Remove a student record permanently.

### 6. Save Data

Stores all records into:

```text
students.txt
```

### 7. Load Data

Automatically loads saved records when the program starts.



